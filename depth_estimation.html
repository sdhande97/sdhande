Depth Estimation from Single Image

NYU DATASET

DENSE NET ARCHITECTURE

DENSENET DETAILS

ENCODER-DECODER ARCHITECTURE

Project Details

  • Category: DEEP LEARNING
  • Libraries: PYTORCH, MATPLOTLIB, SKLEARN
  • Architecture: ENCODER-DECODER ARCH.
  • Pre Trained Model: DENSE NET 169
  • Trained On: RTX 5000 with VRAM of 16Gb
  • Programming Language: PYTHON

Objective: Depth estimation from a single image using encoder-decoder architecture with skip connections and with the help of transfer learning.

The dataset used was NYU Depth dataset which contains the indoor scenes images along with their raw depth maps of size 620x480.

DenseNet as Encoder: In a standard CNN, we have an input image, that is then passed through the network to get an output predicted label in a way where the forward pass is quite straight forward as shown in slide 2 fig 1. The DenseNet architecture is all about modifying this standard CNN architecture as shown in slide 2 fig 2. In this each layer is connected to every other layer, the input of every layer inside DenseNet is the concatenation of feature maps from previous layers.

So in our case, the input RGB image along with depth maps are encoded into a feature vector using the DenseNet-169. The encoded feature vector after bottleneck layer are further fed to upsampling layers of decoder architecture. This decoder layers utilizes the skip connections to take the downsampled filters as input from the same level encoder layers and have 1x1 and 3x3 convolutional layers to reduce the number of filters and to construct the final depth map of half the input resolution that is 310x240.
