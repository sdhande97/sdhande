Project Details

  • Category: DEEP LEARNING
  • Libraries: PYTORCH, TENSORBOARD
  • Trained On: NVIDIA RTX 3060
  • Programming Language: PYTHON

Objective: To perform the automatic Image Generation using the Deep Convolutional Generative Adversarial Network architecture.

DCGANs uses convolutional and convolutional-transpose layers in the generator and discriminator. The generator is able to take random noise and map it into images such that the discriminator cannot tell which images came from the dataset and which images came from the generator.

The generator takes 100x1 random noise as input and expands to 4x4x1024, 8x8x512, 16x16x256, 32x32x128, 64x64x3. All this layers consists of convolutional layers with 5x5 filter size, LeakyRelu as activation function at each layer except the last layer which consist of Tanh as activation function. But a generator alone will just create random noise, conceptually the discriminator in GAN provides guidance to the generator on what images to create. By training on real images and generated images, DCGAN builds a discriminator to learn what feature makes images real.

The discriminator looks at real images and generated images separately, it distinguishes whether the input image to the discriminator is real or generated. The output is the probability that the input is real. We train the discriminator just like a deep network classifier, so that if the input is real the discriminator will output 1 and if the input is generated image then it will be zero. Through this process, the discriminator identifies features that contribute to real images. So, we want the generator to generate the images which can match the real image, so we train generator with backpropagation which takes the target values all the way back to generator which helps generator to create images that towards what the discriminator thinks it is real.

We train both the networks in alternating steps and lock them in fierce competition to improve themselves. Eventually, the discriminator identifies the tiny difference between real and the generated images, and the generator creates images that the discriminator cannot tell the difference. The DCGAN model eventually converges and produces natural looking images.
