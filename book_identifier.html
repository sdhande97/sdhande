Book Name Identifier

DEMONSTRATION

DETECTED KEYPOINTS

MATCHED KEYPOINTS

Project Details

  • Category: COMPUTER VISION
  • Libraries: OPENCV
  • Feature Detector: ORB
  • Programming Language: PYTHON

Objective: To identify name of the book by detecting features on the book in live frame and matching it with the features with other books in list and giving out the name of the book with most matches.

ORB: ORB is a fusion of FAST(Features from Accelerated Segment Test) keypoint detector and BRIEF(Binary Robust Independent Elementary Features) descriptor. First it uses the FAST to find the keypoints, then apply Harris Corner measure to find the top N points among them. After detecting keypoint we go on to compute a descriptor for every one of them. ORB uses BRIEF descriptors, this feature descriptors encode interesting information into a series of numbers and act as a sort of numberical fingerprint that can be used to differentiate one feature from another.

Similarly, here the keypoints and descriptors of the live frame and training image are detected using ORB. Then for matching the computed descriptors KNN classifier is used for classification of various descriptors into the categories in the image and matched by Brute Force Approach. This approach takes the classifed descriptor by KNN from live frame and matches it with all the descriptors in same category in the training image. This extensively reduces the computational time and is very fast and accurate.
