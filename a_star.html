A-Star Algorithm

Project Details

  • Category: ROS
  • Simulator: STAGE: WORLD
  • Software: RVIZ
  • Programming Language: PYTHON

Objective: To perform the navigation of a robot from a given starting point to the destination using the A* algorithm.

Challenge: The main challenge in implementing the A* algorithm is that it requires the occupancy grid of a map. Occupancy grid is a discretization of a map into cells, or in another words itâ€™s a grid representation of a map with 1s and 0s where 1s indicate the obstacle and 0s represent the empty cell.

A* Algorithm: It has 3 elements f, g and h, where g is the lowest cost of the path from current node to the start node and h is the heuristic that estimates the lowest cost from the current node to the target node. Here to calculate the heuristic the Euclidean distance metric was used. And finally, the f is the addition of both the elements g and h. So, at each step the algorithm picks the node which have the lowest value of f.

This is repeated until the shortest path is obtained from the start position to the given goal. Once the path is obtained the set of lowest cost nodes position are converted into coordinates on a map and given to a robot to achieve and reach its goal.
